#  TrendRadar AI: Multi-Agent Market Analysis System

**TrendRadar AI** — это интеллектуальная система глубокого анализа рыночных трендов, построенная на базе **LangChain** и **OpenAI**. Проект реализует многоагентную архитектуру для автономного поиска, обработки данных и визуализации прогнозов.

Система разработана как демонстрация навыков работы с LLM-оркестрацией, асинхронными API и контейнеризацией приложений в рамках профессиональной переподготовки в области программной инженерии.

---

##  Основные возможности
* **Автономный поиск**: Глубокое сканирование сети с использованием специализированных инструментов поиска.
* **Многоагентная аналитика**: Разделение задач на роутинг, исследование, извлечение данных и редактирование для исключения зависаний и повышения точности.
* **Визуализация трендов**: Автоматическая генерация графиков (Matplotlib) на основе извлеченных из текста прогнозных данных за 2024–2026 гг.
* **Forbes-style Отчеты**: Генерация структурированных Markdown-отчетов с кликабельными источниками и профессиональной версткой.

---

##  Архитектура агентов
Процесс обработки запроса разделен на 5 этапов для предотвращения галлюцинаций и обеспечения стабильности:
1.  **Router**: Определяет стратегию обработки (сравнение или суммаризация).
2.  **Researcher**: Собирает технические данные и актуальные рыночные отчеты.
3.  **Analyst**: Формирует глубокий экспертный контент по 5 ключевым секциям.
4.  **Extractor (Visualizer)**: Изолированно извлекает числовые данные и инициирует построение графиков.
5.  **Editor**: Выполняет финальную верстку, добавляет эмодзи и преобразует ссылки в активный Markdown-формат.

## 🛠 Технологический стек
* **Backend**: Python 3.12, FastAPI
* **AI Framework**: LangChain, OpenAI API
* **Data Vis**: Matplotlib (используется неинтерактивный бэкенд `Agg` для стабильной работы в потоках)
* **Testing**: Pytest, HTTPX (ASGITransport)
* **DevOps**: Docker, Docker Compose

##  Инструкция по запуску (Docker)

### 1. Настройка окружения
Создайте в корне проекта файл `.env` и добавьте ваши ключи:
```env
OPENAI_API_KEY=your_openai_api_key
OPENAI_API_BASE=https://api.openai.com/v1
MODEL_ID=qwen/qwen3.6-plus:free

2. Сборка и запуск
Выполните команду в терминале:

docker-compose up --build
После завершения сборки интерфейс будет доступен по адресу: http://localhost:8000

Тестирование
Для проверки стабильности системы и корректности работы всех компонентов (API, Агенты, Визуализатор) запустите тесты внутри контейнера:

# Запуск всех тестов с выводом логов прогресса агентов
docker-compose exec trend-radar pytest -s
